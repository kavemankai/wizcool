# Fringe Ledger — Privacy Policy

_Last updated: 2026-06-11_

Fringe Ledger is a premium, fully offline, single-player game.

## Data collection

**Fringe Ledger does not collect, store, transmit, or share any personal
data.** The game:

- has no account system, login, or registration
- contains no advertising and no third-party analytics SDKs
- makes no network connections during play
- contains no in-app purchases and no user-generated content

## Local data

Game progress (campaign state, settings) is saved **only on your device**,
in the app's private storage. It is removed when the app is uninstalled.
It is never uploaded anywhere.

## Children

Because no data is collected from any user, no data is collected from
children.

## Changes

If a future update changes any of the above, this policy will be updated
and the store listing will link to the revised version before that update
ships.
